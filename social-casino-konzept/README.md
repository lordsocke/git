# Social-Sportsbook-Konzept „ARENA"

Gesamtkonzept für eine native iOS-App mit virtueller Währung (nur Coins) im Auftrag eines internationalen Sportwettenanbieters (Arbeitsannahme: Merkur Bets/Gauselmann) — **Sport im Kern**, ARENA Liga als immer verfügbarer Loop und Coin-Senke, Slots nur als Freispiel-Minigame, Clubs mit Tipp-Duellen.
